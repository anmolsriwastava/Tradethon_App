# QuantDrill

**Live:** [https://quantdrill.onrender.com](https://quantdrill.onrender.com)

A real-time multiplayer trading simulation where players solve probability and Expected Value Puzzles and get the fair market price and trade against other players and bots (Created to provide liquidity in market). 

---

## 🎮 How It Works

1. **Join a room** with friends
2. **Solve a probability puzzle** (biased coin, dice games, etc.)
3. **Find the fair value** using expected value math
4. **Trade** in a live order book against AI bots
5. **5 rounds** — highest PnL wins

---

## 🤖 Bot Personalities

| Bot Type | Behavior |
|----------|----------|
| **Market Maker** | Posts bid/ask spreads, provides liquidity |
| **Noise Trader** | Trades randomly, creates volatility |
| **Informed Trader** | Knows ~70% of true value, trades intelligently |
